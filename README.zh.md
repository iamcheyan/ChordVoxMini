---

### 关于这个版本

**ChordVoxMini** 并不是一个全新的项目，而是在优秀的开源项目 ChordVox 基础上，根据个人需求进行了一些细节上的小修改：

1. **简化逻辑**：尝试移除了原版中的 Pro 订阅、账户登录及付费墙相关代码。
2. **离线优先**：坚持让所有的语音识别（STT）和文本处理都在本地完成，不依赖云端 API。
3. **新增翻译小尝试**：尝试加入了一个简单的中日离线翻译功能，方便说中文直接出日文。

---

### 主要功能

- ⌨️ **语音输入**：一键触发录音，识别后自动粘贴。
- 🧠 **本地润色**：利用本地小模型修正语气词，让输出更整洁。
- 🌐 **离线中日翻译**：实验性功能。支持中文语音输入，本地转换为日语，适合简单的跨语言交流。
- 🚀 **Mac 优化尝试**：
    *   针对 Apple Silicon 芯片尝试开启了 **CoreML 加速**。
    *   优化了进程常驻逻辑，尽量缩短翻译的等待时间。

---

### ⚠️ 一点说明

- **稳定性**：因为改动得比较匆忙，目前可能存在不少 Bug 或不稳定现象。
- **跨平台**：目前主要在 **macOS** 平台上自用。Windows 和 Linux 版暂时没有精力去做完整的适配和测试，可能需要大家自行调试。
- **技术支持**：这只是一个个人维护的分支，精力有限，无法提供即时的技术支持，遇到问题可能需要大家多多包涵。

---

### 开发与编译

1. **环境准备**：需要 Node.js 和 Python 3。
2. **安装依赖**：`npm install`
3. **启动开发模式**：`npm run dev`
