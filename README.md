### 项目初衷

**ChordVoxMini** 是基于原版 ChordVox 进行了一些小改动的维护版本。我们非常喜欢原版的设计理念，并在此基础上做了一些更符合个人需求的调整：

1. **移除付费相关逻辑**：尝试去掉了原版中的订阅和付费校验部分，让所有功能回归本地。
2. **坚持离线优先**：我们希望所有的语音和文本处理都能在用户自己的电脑上完成。
3. **增加离线翻译小功能**：尝试集成了一个中日离线翻译引擎，支持在本地将中文语音转换为日语文字输入。

---

### 主要功能

- ⌨️ **语音输入**：支持录音、识别并自动粘贴到光标处。
- 🧠 **本地精修**：通过本地模型自动修正口语中的语气词和标点。
- 🌐 **离线中日翻译**：一个实验性功能，支持中文说出，日语输入，全程本地推理。
- 🔒 **隐私保护**：不强制联网，不上传数据，音频处理均在本地。

---

### 针对 Mac 的小优化

我们为 Mac 用户（尤其是 Apple Silicon 芯片）做了一些尝试：
- **翻译硬件加速**：翻译引擎尝试接入了 CoreML 加速，希望能在提升速度的同时降低一点功耗。
- **响应速度优化**：通过优化加载逻辑，让翻译响应尽可能快一些。

---

### ⚠️ 碎碎念与提醒

- **关于质量**：修改得比较匆忙，代码里可能还藏着不少 Bug，稳定性肯定不如原版。
- **关于兼容性**：目前主要在 macOS 上自己用，Windows 和 Linux 还没来得及做详细的跨平台测试，可能需要大家多担待或自行调试。
- **关于支持**：这只是一个兴趣驱动的小改动版，我们没有能力提供专业支持，遇到问题可能需要大家互相帮助。

---

### 如何开始

1. **安装环境**：确保安装了 Node.js 和 Python 3。
2. **安装依赖**：`npm install`
3. **启动项目**：`npm run dev`
