# ChordVoxMini

ChordVoxMini 是一个基于开源项目 ChordVox 进行小规模修改的维护版本。本项目的主要目标是提供一个纯粹、离线且免费的语音输入工具。

### 主要特点

1. 离线使用：移除了原版中的所有付费订阅和在线账户逻辑，确保所有语音识别和文本处理均在本地完成。
2. 隐私保护：不强制联网，不收集任何用户音频或文本数据。
3. 离线翻译：集成了一个实验性的中日离线翻译功能，支持通过语音直接输入翻译后的日语。
4. 硬件加速：针对 Mac 设备（Apple Silicon 芯片）尝试开启了 CoreML 加速，以优化翻译推理速度和能耗。

### 核心功能

* 语音输入：通过快捷键触发录音，识别后自动粘贴至光标位置。
* 文本润色：利用本地模型修正口语习惯，使输出内容更符合书面表达。
* 中日翻译：支持将中文语音实时转换为日语文字并粘贴。

### 注意事项

* 开发说明：本项目由个人在较短时间内完成修改，可能存在较多不稳定因素和 Bug。
* 平台支持：目前仅在 macOS 平台上进行了初步测试，Windows 和 Linux 版本的兼容性尚未验证。
* 技术支持：作为一个个人维护的修改版，我们无法提供即时的技术支持，请用户根据实际情况自行调试或解决问题。

### 开发与运行

1. 安装 Node.js 和 Python 3 环境。
2. 在项目根目录运行 `npm install` 安装依赖。
3. 运行 `npm run dev` 启动开发模式。
